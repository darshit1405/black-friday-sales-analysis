# 🛒 Black Friday Sales Analysis

This project focuses on analyzing customer purchase behavior using the Black Friday dataset.

---

## 📌 Objective

To identify key factors that influence customer spending during Black Friday sales.

---

## 📊 Dataset

* Source: Black Friday dataset
* Contains information about:

  * User demographics
  * Product categories
  * Purchase amounts

---

## 🧠 Key Analysis Performed

### 🔹 Data Cleaning

* Handled missing values
* Checked data types
* Removed inconsistencies

---

### 🔹 Exploratory Data Analysis (EDA)

#### 1. Gender Analysis

* Compared purchase behavior between male and female customers

#### 2. Age & Marital Status

* Analyzed how age groups and marital status affect spending

#### 3. Occupation & Product Analysis

* Identified which occupations spend more
* Studied popular product categories

#### 4. Multi-column Analysis

* Combined multiple features to find deeper insights

---

## 📈 Key Insights

* Male customers generally have higher purchase amounts
* Certain age groups contribute more to sales
* Occupation plays a significant role in spending behavior
* Product categories influence purchasing trends
